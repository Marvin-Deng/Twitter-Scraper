# DIG-120

## Usage

### Setup

```shell
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Running

1. Log into [Twitter](https://x.com/home)

2. Run script

```shell
source venv/bin/activate
python3 src/main.py
```

### Update Requirements

```shell
pip freeze > requirements.txt
```

### Formatter

```shell
black .
```
